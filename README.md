#### 安装教程

1.  git remote set-url origin https://gitee.com/kikoblog/ZLX.git	更换远程仓库地址
2.  git pull 拉取最新文件 push之前输入
3.  git add . 提交到暂存区
4.  git commit -m "" 提交到本地仓库
5.  git push 上传到远程仓库
